# AI Usage Statement

## AI Tool

OpenAI ChatGPT (GPT-5.5)

---------------------------------------------------------------------------------------------

## Purpose

AI was used as a development assistant during the implementation of this project.

The assistance focused on improving software design, code quality, documentation, and explanations of Python concepts.

---------------------------------------------------------------------------------------------

## How AI Was Used

AI assistance included:

- Explaining project requirements
- Discussing software architecture
- Designing the project structure
- Reviewing implementation approaches
- Suggesting modular function design
- Explaining Python libraries (Pandas, Folium, Geopy, Jinja2)
- Improving code readability
- Assisting with documentation
- Assisting README and project documentation
- Providing debugging suggestions during development

---------------------------------------------------------------------------------------------

## Human Contribution

The project implementation involved:

- Understanding the assessment requirements
- Making architectural and implementation decisions
- Writing, integrating, testing, and validating the application
- Debugging issues during development
- Verifying outputs using the provided datasets
- Reviewing and refining the generated code

---------------------------------------------------------------------------------------------

## Verification

All generated code was reviewed, integrated, tested, and validated before being included in the final project.

The final submission represents a working application that was verified using the supplied datasets.
